# Retail-Orders-Analytics-Pipeline-Project
End-to-End ETL + SQL Analytics | Python, pandas, SQL Server

**End-to-End ETL + SQL Analytics Project**  
*Python → pandas → SQL Server → Business Insights*

## Business Problem Solved

-Manual retail order processing took 4+ hours daily. Built automated pipeline delivering actionable insights.

##  Tech Stack

-python (pandas, kaggle, sqlalchemy) | SQL Server | Regex | Window Functions (ROW_NUMBER)


##  What This Project Does

1️⃣ Kaggle API → Downloads 25K+ retail orders

2️⃣ Pandas ETL → Cleans nulls, engineers profit column

3️⃣ SQL Server → Optimized table structure

4️⃣ 6 SQL Queries → Revenue, growth, peak analysis


##  Key Business Insights

- **80% YoY profit growth:** Supplies subcategory
- **14% sales drop:** South region post-festival
- **Peak months identified:** Category-wise inventory planning
